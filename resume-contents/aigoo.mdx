FE 3명, BE 2명, UI/UX 1명

::: tip K-pop을 좋아하는 글로벌 팬이 손편지를 통해 아티스트와 소통하고 AI를 통해 한글을 공부할 수 있도록 맞춤형 학습자료를 제공하는 서비스

::: important Epson Innovation Challenge 참가 프로젝트

##### 서비스 주요 기능

- Epson 복합기를 사용한 손편지 인쇄 및 스캔 업로드
- 아티스트와 손편지를 주고 받을 수 있는 편지함 제공
- 업로드 된 편지를 **Google Gemini AI**가 문장단위로 해석해 원문/한글로 분리 및 키워드 추출
- 해석된 편지에서 원하는 키워드를 선택해 맞춤형 학습자료 생성
- Epson 복합기를 사용한 학습자료 인쇄

##### 사용 기술

- `Next.js`, `Typescript`, `Tailwind CSS`, `Zustand`, `React Query`

##### 맡은 역할

::: note 프로젝트 팀장, 손편지에서 키워드를 선택하는 일련의 페이지 제작

- **손편지 상세 보기 페이지**

  - 정규표현식과 `exec`를 활용해 문자열 파싱
  - 문장 내에 포함된 키워드 중 선택된 키워드를 구분해 하이라이트 스타일 지정
  - 선택된 키워드를 서버로 전달

- **문장 단위 키워드 선택 페이지 제작**
  - Open AI의 chatGPT API를 활용해 각 키워드별 번역, 용례, 반의어, 동의어 등의 데이터 적용
  - Next.js의 **Route Handler**로 API key의 클라이언트 노출 문제 해결

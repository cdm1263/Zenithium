FE 3명

::: tip 인기 게임 포켓몬스터의 도감 및 커뮤니티 서비스

##### 서비스 주요 기능

- 타입별 포켓몬 모아보기
- 포켓몬 상세 스펙 및 진화 과정 정보 제공
- 포켓몬 찜하기 표시
- 커뮤니티 글, 댓글 작성
- 개인 맞춤형 포켓몬 카드 제작

##### 사용 기술

- `Next.js`, `Typescript`, `SCSS`, `Zustand`, `React Query`

##### 맡은 역할

::: note 프로젝트 기획, 메인 도감 페이지 및 개인 카드 제작 페이지 제작

- **메인 도감 페이지 제작**

  - 검색 및 타입 필터로 원하는 포켓몬을 고를 수 있도록 필터링 로직 구성
  - 1000개가 넘는 데이터 호출을 위해 **batch size** 개념 도입. 분할 비동기 호출 구현
  - **windowing** 으로 현재 뷰포트에 해당하는 UI만 렌더링. 초기 로드시간 **약 73%** 향상

- **개인 카드 제작 페이지 제작**

  - 하나의 input에서 포켓몬 이름과 타입을 함께 검색할 수 있는 기능 제공
  - 찜한 포켓몬, 랜덤 포켓몬에서 포켓몬을 선택할 수 있는 기능 제공
  - 유저가 직접 포켓몬 이름을 지어 카드를 제작할 수 있도록 구성
